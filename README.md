Download this for WINDOWS: Exile2.builder_0-5-1_vA3_Windows.exe

This app was created while trying to help ChatGPT understand the Passive Tree through uploading the data.json file on GGG's github.

It evolved into a Passive Tree Builder and ".build" exporter.

This app can do the following:
- Sort highlight based on overlapping drop downs catagorized by Weapon, Damage Type, Defense and even Skill
- The highlighted nodes are directly relevent to a build using the selected drop down option
- The app can Import and Export ".BUILD" files used for in-game build guides since PoE2 0.5 patch
- The app can directly Export ".BUILD" files into the PoE2 "BuildPlanner" folder location to immediately be used in-game.

I'm sharing it because, I like it. I didn't make it. I had the idea, and ChatGPT made it. There are similar things out there. This is self contained and not connected to a login.
